Principais formatações

Negrito: Destaque

Nesse texto, temos uma frase em negrito utilizando a tag "b"

Já nesse texto, tem uma frase em negrito utilizando a tag "strong"

Itálico: ênfase

Nesse texto, temos uma frase em itálico, utilizando a tag "i"

Já nesse texto, temos uma frase em itálico, utilizando a tag "em"

Texto grifado

Nesse texto, temos uma frase marcada, utilizando a tag "mark"

Texto sublinhado

Temos um texto sublinhado com a tag "ins"

Mas também podemos sublinhar esse texto com a tag "u", que é antiga

Texto deletado

Para este caso, para desconsiderar uma frase no texto, utilizamos a tag "del"

Texto sobrescrito

Para textos do tipo fórmula matemática, como por exemplo: x-2 + 5, utiliza-se a tag "sup"

Texto subscrito

Para textos do tipo fórmula química, como por exemplo: H2O, utiliza-se a tag "sub"
